# Sytem-fenix# Sistema de Gestão de Clientes

Sistema completo para gestão de clientes com controle de pagamentos, pastas e histórico de acessos.

## 🎨 Cores do Sistema
- **Preto** (#000000) - Fundo, elementos de destaque
- **Azul** (#0056b3) - Header, botões principais
- **Laranja** (#ff6b00) - Destaques, ícones, ações importantes
- **Branco** (#FFFFFF) - Fundo do conteúdo

## ✨ Funcionalidades

### 📊 Dashboard
- Total de clientes cadastrados
- Pagamentos realizados
- Clientes pendentes
- Valor total em aberto

### 👥 Gestão de Clientes
- ✅ Cadastro completo
- ✅ Edição de dados
- ✅ Exclusão de registros
- ✅ Visualização detalhada
- ✅ Busca por nome
- ✅ Filtros por status e pagamento

### 💰 Controle Financeiro
- Valor total do serviço
- Valor já pago
- Saldo devedor
- Status de pagamento (Pago/Parcial/Pendente)

### 📁 Controle de Pastas
- Indicação se a pasta foi feita
- Registro de quem pegou a pasta
- Histórico de acessos

### 🔐 Segurança
- Sistema de login
- Cadastro de novos usuários
- Registro de quem acessou cada cliente

### 📤 Importação/Exportação
- Exportar dados em JSON
- Importar dados de backup
- Limpar todos os dados

## 🚀 Como Usar

### 1. Primeiro Acesso
1. Abra o arquivo `login.html`
2. Use as credenciais padrão:
   - Usuário: `admin`
   - Senha: `admin123`
3. Cadastre novos usuários se necessário

### 2. Gerenciar Clientes
- **Adicionar:** Preencha o formulário e clique em "Salvar Cliente"
- **Editar:** Clique no botão "Editar" no card do cliente
- **Excluir:** Clique em "Excluir" e confirme
- **Visualizar:** Clique em "Visualizar" para ver todos os detalhes

### 3. Registrar Acessos
- Clique em "Registrar Acesso" para marcar quem pegou a pasta
- O sistema registra automaticamente o usuário e horário

### 4. Backup dos Dados
- Use "Exportar Dados" para fazer backup
- Use "Importar Dados" para restaurar backup
- Os dados são salvos no navegador automaticamente
